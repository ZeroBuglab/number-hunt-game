# 1.Random қалай жұмыс істейді?  
Python тіліндегі random модулі псевдокездейсоқ сандарды генерациялайды.  
Яғни компьютер шынайы кездейсоқ сан шығармайды, ол арнайы алгоритм қолданады — Mersenne Twister.  
Бұл алгоритм бастапқы мәнді (seed) алып, формула арқылы келесі сандарды есептейді.

# 2. Цикл қалай ұйымдастырылған?  

Ойын while циклі арқылы жұмыс істейді.

Цикл:
* әрекет қайталанады  
* әр қадамда әрекет (пайдаланушы енгізуі) тексеріледі  
* әрекеттер саны азаяды  
** Егер ойыншы дұрыс тапса → break арқылы цикл тоқтайды

# 3. Подсказка логикасы   
Подсказка қарапайым салыстыру арқылы жасалады:     
1. егер сан үлкен болса → “тым үлкен”  
2. егер сан кіші болса → “тым кіші”  
3. Бұл if / elif шарттары арқылы жүзеге асады.

# 4. Деңгейлерді қалай қосамыз?    
Деңгей диапазон арқылы анықталады:      
1. Жеңіл: 1–10    
2. Орта: 10–100  
3. Қиын: 100–1000   
Пайдаланушы таңдаған деңгейге байланысты кездейсоқ санның шекарасы өзгереді.

# 6. “Тоқтату / Жалғастыру” батырмасы қайда?    
** Бұл функция пайдаланушы енгізуі арқылы жасалады:    
** yes → жалғастыру  
** no → тоқтату

# 7. Қиындық рейтингі
